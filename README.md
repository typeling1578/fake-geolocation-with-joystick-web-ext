## How to build

### 1. Install packages

```bash
pnpm install --frozen-lockfile
```

### 2. Build

#### Chrome

```bash
pnpm run zip
```

Output path: `./.output/fake-geolocation-with-joystick-*.*.*-chrome.zip`

#### Firefox

```bash
pnpm run zip:firefox
```

Output path: `./.output/fake-geolocation-with-joystick-*.*.*-firefox.zip`
